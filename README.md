# QuikShot

A simple screenshot utility for Windows 10 and later.
It's designed to make it easy and fast to take mutltiple screenshots into a single directory.

## How to use
1. Open the app.
2. Select a destination directory.
3. Press `ALT+SHIFT+S` to begin.
4. Press `ALT+SHIFT+T` to take a screenshot.
5. Fill out file name.
6. Press enter.
7. Repeat.
